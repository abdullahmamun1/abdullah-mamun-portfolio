# Abdullah Mamun — Portfolio

## Description
A personal portfolio website for Abdullah Mamun, a Full Stack Developer and
Computer Science & Engineering student at SUST, Sylhet. The site showcases
his profile, skills, education, experience, and projects, with a responsive
layout and a light/dark theme toggle.

## Tech Stack
- **Next.js** (App Router) — React framework
- **TypeScript**
- **Tailwind CSS v4** — styling
- **Framer Motion** — animations and interactions
- **next-themes** — light/dark mode
- **lucide-react** — icons

## How to Run

```bash
# 1. Install dependencies
npm install

# 2. Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Other commands
```bash
npm run build   # production build
npm run start   # run the production build
npm run lint    # run eslint
```

## Edit content
All personal content (name, bio, skills, education, experience, projects,
contact info, social links) lives in `src/data/profile.ts`. Update that file
to customize the site.
